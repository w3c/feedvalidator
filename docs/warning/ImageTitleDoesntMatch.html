Feed Validation Service

Check the syntax of Atom or RSS feeds

Message

Image title doesn't match channel title

Explanation

According to the RSS Advisory Board's Best Practices Profile, this element should have the same text as the channel's title element

Solution

Repeat the channel's title element in the image.
