Feed Validation Service

Check the syntax of Atom or RSS feeds

Message

Ensure description precedes content:encoded

Explanation

While either the description or the content:encoded elements may contain the full content of an item, the description element is more widely supported than the content:encoded element in the context of an RSS 2.0 feed.

When both elements appear in an RSS 2.0 item, some consumers will simply pick up the last such element.

See the RSS Advisory Board's Best Practices Profile for additional requirements and recommendations.

Solution

Publishers who employ summaries should store the summary in description and the full content in content:encoded, ordering description first within the item. On items with no summary, the full content should be stored in description.
