Feed Validation Service

Check the syntax of Atom or RSS feeds

Message

foo link relation found in entry

Explanation

RFC 5005 defines these relations in the context of a "Head section" (e.g., the child elements of the atom:feed element in an Atom Feed Document) but in this feed, archive and/or history links were found outside of the head section.

Solution

If these links are intended to express feed paging or archiving relationship, move the link elements themselves outside of any entry.
