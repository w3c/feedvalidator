Message

Use zero for unknown length

Explanation

This value must be an integer greater than or equal to zero. It can not be negative, or a decimal, or a fraction, or a string, or blank.

Solution

According to the RSS Advisory Board's Best Practices Profile, when an enclosure's size cannot be determined, a publisher should use a length of 0.
